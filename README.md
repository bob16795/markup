# markup - A Markup Impersonation 

```text
markup --help|-h
markup --version|-v
markup --compile|-c
```


`--help`|`-h`  
Show help and exit.

`--version`|`-v`  
Show version and exit.

`--compile`|`-c`  
Compiles a document
